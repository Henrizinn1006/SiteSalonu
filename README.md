# Salonu — Site Institucional

Site institucional desenvolvido para apresentação do método 7PCG (Criação Gastronômica com Sistema), com foco em conversão, design moderno e experiência do usuário.

---

## Sobre o projeto

Este projeto foi desenvolvido do zero utilizando HTML, CSS e JavaScript, com foco em:

- Design moderno (dark mode)
- Estrutura profissional
- Responsividade (desktop e mobile)
- Conversão (CTAs estratégicos)
- Performance e simplicidade

O site apresenta:

- Método 7PCG
- Livros e materiais
- Serviços para empresas
- Chamadas para ação (compra e contato)

---

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Hospedagem: Hostinger

## Contato integrado

Os botões de ação direcionam para envio de e-mail com mensagem pré-definida.

Exemplo:

```text
mailto:contato@salonu.com.br?subject=Solicitação de Serviço&body=Olá, gostaria de solicitar...
```

## Deploy

O projeto está hospedado na Hostinger, utilizando:

- Arquivos dentro da pasta public_html
